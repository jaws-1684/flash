# Flash 
<a href="https://flash.jawsprojects.net/users/sign_in">View Demo</a>

## Description
A real-time **messenger** application built with Ruby on Rails + Inertia[^1]
## About The Project
<p align="center">
  <img height="350" src="https://raw.githubusercontent.com/jaws-1684/flash/refs/heads/images/165525.png">
</p>
This app lets its signed in users connect with their friends through messages.

## Features
- **Dark and light** mode
- **Search** through chats, groups and users 
- **Authentication** system
- **Send, edit, delete** messages 
- **Create, delete** chats and groups
- **Join, Leave** groups
- **Image uploads** attached to messages

## Technologies Used

### Backend
- **Ruby on Rails** - Full-stack web framework
- **Inertia**
- **PostgreSQL** - Primary database
- **Postgres search** - PostgreSQL text search
- **Devise** - Authentication and authorization
- **OAuth** - GitHub and Google social login integration

### Frontend
- **Tailwind CSS** 
- **React** 

### File Storage
- **Cloudinary** - Cloud storage for image uploads

## Getting Started

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/jaws-1684/flash.git
   ```
2. Navigate into the project directory
   ```sh
   cd flash
   ```
3. Install dependencies
   ```sh
   bundle install
   ```
